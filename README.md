# GetOpticalMaps

Este proyecto está actualmente incompleto y se está desarrollando para solucionar una problemática personal. me di cuenta de que me faltaba una forma efectiva de promocionar mis productos. Esta herramienta, GetOpticalMaps, busca ópticas donde extrae la info y de esta forma contactarlos para mejorar la visibilidad y el alcance de mis ventas.

## Estado del Proyecto

- **Incompleto**: El proyecto aún no está terminado y puede contener errores o funcionalidades faltantes.
- **Propósito**: Solucionar la falta de promoción en el negocio de venta de marcos mediante herramientas automatizadas.

## Instalación

1. Clona este repositorio.
2. Crea un entorno virtual: `python -m venv env`
3. Activa el entorno: `env\Scripts\activate` (en Windows)
4. Instala las dependencias: `pip install -r requirements.txt` (si existe, de lo contrario instala manualmente las necesarias como Selenium)

## Uso

Ejecuta el script principal: `python src/main.py`

Nota: Como el proyecto está incompleto, el uso puede variar y no todas las funcionalidades están implementadas.
